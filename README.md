React + Vite

Este template fornece uma configuração mínima para fazer o React funcionar no Vite com HMR e algumas regras do ESLint.

Atualmente, dois plugins oficiais estão disponíveis:

@vitejs/plugin-react
 usa Oxc
@vitejs/plugin-react-swc
 usa SWC
React Compiler

O React Compiler não está habilitado neste template por causa do impacto no desempenho durante o desenvolvimento e a build. Para adicioná-lo, consulte esta documentação
.

Expandindo a configuração do ESLint

Se você está desenvolvendo uma aplicação para produção, recomendamos usar TypeScript com regras de lint baseadas em tipos habilitadas. Consulte o template TS
 para obter informações sobre como integrar TypeScript e typescript-eslint
 ao seu projeto.
